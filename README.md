### 1. Дана грамматика. Постройте вывод заданной цепочки.
a) S : T | T '+' S | T '-' S; \
   T : F | F '*' T; \
   F : 'a' | 'b';

Цепочка: a - b * a + b

<b>Ответ:</b>

``S -> T - S`` \
``-> F - T + S`` \
``-> a - F * T + T`` \
``-> a - b * F + F`` \
``-> a - F * T + T`` \
``-> a - b * a + b``
  
b) S : 'a' S B C | 'ab' C; \
   C B : B C; \
  'b' B : 'bb'; \
  'b' C : 'bc' ; \
  'c' C : 'cc';
  
<b>Ответ:</b>

``S -> aSBC`` \
``-> aaSBCBC`` \
<code>-> aaab<b>CBCB</b>C</code> <-- Меняем все CB на BC \
<code>-> aaabB<b>CB</b>CC</code> <-- Оставшуюся CB меняем на BC \
``-> aaabBBCCC`` \
``-> aaabbBCCC`` \
``-> aaabbbCCC`` \
``-> aaabbbcCC`` <-- 'b'C : 'bc' \
``-> aaabbbccc``

  '0' A : '00' A '1' \
  ^^ \
  вот он </i>
  
b)S : A 'b'; \
A : A 'a' | 'ba';

<b>Ответ:</b>

<i>Регулярная. В первой строчке S : A 'b', где S,A ∈ VN, 'b' ∈ VT*; \
